# SnapReview 개발 로드맵

> 마지막 업데이트: 2026-04-08
> 버전: v1.0

---

## 프로젝트 개요

영수증 촬영/업로드만으로 AI가 리뷰 문장을 자동 생성하는 서비스. Google Vision AI로 영수증 정보를 추출하고 Gemini API로 짧게/보통/상세 3가지 버전의 리뷰를 자동 생성하여, 한국인 소비자가 네이버·카카오·구글 등 플랫폼에 리뷰를 남기는 데 드는 시간과 노력을 제거한다.

---

## 전체 로드맵 개요

```
Phase 0 (1주)       Phase 1 (1.5주)      Phase 2 (2주)        Phase 3 (1주)
UI 프로토타입    →   더미 데이터 개발   →   실제 API 연동     →   완성 및 배포
─────────────────   ─────────────────   ─────────────────   ─────────────────
정적 화면 4개       Mock 데이터         Google Vision AI     에러 처리 보완
레이아웃/반응형      Zustand 상태관리    Gemini API           엣지 케이스 처리
사용자 플로우 확인   인터랙션 구현       Supabase 연동        Vercel 배포
                    LocalStorage CRUD   API Route 구현       성능 최적화
```

**총 예상 기간: 5.5주**

- 모든 에러 시나리오(네트워크 오류, API 한도 초과, 파일 형식 오류 등)에서 적절한 안내가 표시된다

| 스타일링 | TailwindCSS v4 + shadcn/ui | 모바일 퍼스트 반응형 빠른 구현, 접근성 기본 제공 컴포넌트 |
| 폼/검증 | React Hook Form 7.x + Zod | OCR 수정 폼 및 파일 입력 검증 |
| OCR | Google Vision AI | 한국어 영수증 인식 정확도, 무료 티어(월 1,000건) |
| AI 생성 | Gemini 2.0 Flash | 한국어 리뷰 생성 품질, 무료 티어 제공 |
| DB/Storage | Supabase | 이미지 저장소 + PostgreSQL, 무료 티어 |
| 상태관리 | Zustand + LocalStorage | 단순한 클라이언트 상태, 인증 없이 로컬 히스토리 관리 |
| 배포 | Vercel | Next.js 최적화 배포, 무료 티어 |
