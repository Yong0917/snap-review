# SnapReview 개발 로드맵

> 마지막 업데이트: 2026-04-08
> 버전: v1.5

---

## 프로젝트 개요

음식, 공간, 제품, 소품 등 다양한 사진을 업로드하면 AI가 리뷰 문장을 자동 생성하는 서비스. Gemini Vision이 이미지의 핵심 대상과 분위기를 분석하고 짧게/보통/상세 3가지 버전의 리뷰를 생성하여, 사용자가 네이버·카카오·구글 등 플랫폼에 리뷰를 빠르게 남길 수 있도록 돕는다.

---

## 전체 로드맵 개요

```
Phase 0 (1주)       Phase 1 (1.5주)      Phase 2 (2주)        Phase 3 (1주)
UI 프로토타입    →   더미 데이터 개발   →   실제 API 연동     →   완성 및 배포
─────────────────   ─────────────────   ─────────────────   ─────────────────
정적 화면 4개       Mock 데이터         Gemini Vision        에러 처리 보완
레이아웃/반응형      Zustand 상태관리    Gemini API           엣지 케이스 처리
사용자 플로우 확인   인터랙션 구현       Supabase 연동        Vercel 배포
                    LocalStorage CRUD   API Route 구현       성능 최적화
```

**총 예상 기간: 5.5주**

## 기술 스택 요약

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| 프레임워크 | Next.js 15 (App Router) + React 19 + TypeScript | API Routes로 서버 로직 처리, 무인증 단순 아키텍처에 적합 |
| 스타일링 | TailwindCSS v4 + shadcn/ui | 모바일 퍼스트 반응형 빠른 구현, 접근성 기본 제공 컴포넌트 |
| 폼/검증 | React Hook Form 7.x + Zod | 이미지 분석 정보 수정 폼 및 파일 입력 검증 |
| 이미지 분석/AI 생성 | Gemini 2.5 Flash | 범용 이미지 이해 + 한국어 리뷰 생성 |
| DB/Storage | Supabase | 이미지 저장소 + PostgreSQL, 무료 티어 |
| 상태관리 | Zustand + LocalStorage | 단순한 클라이언트 상태, 인증 없이 로컬 히스토리 관리 |
| 배포 | Vercel | Next.js 최적화 배포, 무료 티어 |
| 패키지 관리 | npm | PRD 명시 |
